# Changelog

All notable changes to the **Chat Session Workflow Viewer** extension will be documented in this file.

## [0.1.0] - 2025-02-21

### Added

- Session tree view in the activity bar listing chat sessions
- Interactive workflow diagram powered by React Flow (user prompts, LLM responses, tool calls, sub-agents, thinking blocks)
- Session metadata panel with timing, model info, and tool usage summary
- Workspace-scoped and global session browsing
- Search and filter sessions by title or message content
- Support for `.json` and `.jsonl` (mutation log) session formats
- Auto-refresh on file changes
- Open raw session file command
